# Terraria中文Wiki语言包加载器

- 作者: Cai  
- 仓库: [github](https://github.com/ACaiCat/WikiLangPackLoader)  
- 启动服务器后自动设置中文，并加载**官方 Wiki 中文语言包**以修正中文
- eg. 克劳得(官方翻译) => 云(Wiki翻译)
- 注: 使用本插件可能导致一些RPG等的给物品、BUFF命令失效
